Expand description

A trait, when implemented must take care of collecting orders (invest & redeem) for a given investment class. When being asked it must return the current orders and when being singled about a fulfillment, it must act accordingly.

fn process_invest_orders( asset_id: Self::InvestmentId ) -> Result<Self::Orders, Self::Error>

When called the manager return the current invest orders for the given investment class. Callers of this method can expect that the returned orders equal the returned orders from invest_orders.

NOTE: Once this is called, the OrderManager is expected to start a new round of orders and return an error if this method is to be called again before invest_fulfillment is called.
