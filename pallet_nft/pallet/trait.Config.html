Expand description

NFT pallet’s configuration trait.

Associated types and constants are declared in this trait. If the pallet depends on other super-traits, the latter must be added to this trait, such as, in this case, [pallet_balances::Config] super-traits. Note that [frame_system::Config] must always be included.
