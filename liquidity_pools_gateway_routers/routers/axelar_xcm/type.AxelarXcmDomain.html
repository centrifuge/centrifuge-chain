Fields§

§location: Box<VersionedMultiLocation, Global>

The XCM multilocation of the domain.

§ethereum_xcm_transact_call_index: BoundedVec<u8, ConstU32<65536>>

The ethereum_xcm::Call::transact call index on a given domain. It should contain the pallet index + the transact call index, to which we will append the eth_tx param.

You can obtain this value by building an ethereum_xcm::transact call with Polkadot JS on the target chain.

§contract_address: H160

The target contract address on a given domain.

§max_gas_limit: u64

The max gas limit for the execution of the EVM call

§transact_required_weight_at_most: Weight

The max weight we want to pay for the execution of the Transact call in the destination chain

§overall_weight: Weight

The overall max weight we want to pay for the whole XCM message (i.e, all the instructions)

§fee_currency: <T as Config>::CurrencyId

The currency in which execution fees will be paid on

§fee_amount: u128

The fee we use to buy execution for the execution of the Transact call
